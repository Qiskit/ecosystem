A Qiskit transpiler plugin implements the plugin interface that allows to hook non-qiskit 
code into the Qiskit transpilation pipeline.
If a project is in this category, consider tagging it with one of these labels:
[`transpiler stage`](../labels#transpiler-stage),
[`unitary synthesis`](../labels#unitary-synthesis), and
[`high-level synthesis`](../labels#high-level-synthesis).

Tutorial:
[Create a Qiskit plugin](https://quantum.cloud.ibm.com/docs/en/guides/create-transpiler-plugin)
