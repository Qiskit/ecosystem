In general, a Qiskit transpiler plugin implements the plugin interface that allows to hook non-qiskit  code into the Qiskit transpilation pipeline.

A project in this category implements a particular flavor of these plugins: [transpiler stage plugin](https://quantum.cloud.ibm.com/docs/en/api/qiskit/transpiler_plugins).

For other kind of transpiler plugins, check [Unitary synthesis plugin](#unitary-synthesis-plugin) and [High-level synthesis plugin](#high-level-synthesis-plugin).

Tutorial:
[Create a Qiskit plugin](https://quantum.cloud.ibm.com/docs/en/guides/create-transpiler-plugin)
