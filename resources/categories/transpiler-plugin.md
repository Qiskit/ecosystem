In general, a Qiskit transpiler plugin implements the plugin interface that allows to hook non-qiskit  code into the Qiskit transpilation pipeline.

 * [transpiler stage plugin](https://quantum.cloud.ibm.com/docs/en/api/qiskit/transpiler_plugins).
 * [unitary synthesis plugin](https://quantum.cloud.ibm.com/docs/en/api/qiskit/qiskit.transpiler.passes.synthesis.plugin.UnitarySynthesisPlugin)
 * [High-level synthesis plugin](https://quantum.cloud.ibm.com/docs/en/api/qiskit/qiskit.transpiler.passes.synthesis.plugin.HighLevelSynthesisPlugin).
